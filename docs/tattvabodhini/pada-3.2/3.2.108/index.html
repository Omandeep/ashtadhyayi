TATTVABODHINIS

January 1, 0001

निषेदुपीमिति। षद्लृ विशरणादौ। लिटि धतोरिति द्वित्वे अत एकहल्मध्ये इत्येत्त्वाभ्यासलोपौ। उगितश्चेति ङीपि वसोरिति सम्प्रसारमए वलादित्वाऽभावान्नेट्। सदिरप्रतेरिति षत्वम्। पुंसि तु निषेदिवान्। अध्यूषुष इति। वस निवासे अधिपूर्वः। वचिस्वपीति संप्रसारणं। द्वित्वहलादिशेषसवर्णदीर्घाः। ङसि भत्वात्संप्रसारणम्। प्रथमैकवचने तु अध्यूषिवान्।
