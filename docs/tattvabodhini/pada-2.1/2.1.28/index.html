TATTVABODHINIS

January 1, 0001

कालाः। बहुवचननिर्देशः स्वरूपनिरासार्थः। कालवाचिनो द्वितीयान्ताः क्तान्तेन सह वा समस्यन्त इति सूत्रार्थः। ननु काला अत्यन्तसंयोगेइत्येवास्तु क्तेनेति निवृत्तम्, नार्थो योगविभागेनेत्यत आह–अनत्यन्तेति। मासप्रमति इति। माङ्माने। आदिकर्मणि क्तः कर्तरि। इह प्रतिपञ्चन्द्रेण नास्त्यत्यन्तसंयोगः।
