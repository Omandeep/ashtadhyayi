परम्परा प्रसज्येत। षष्ठे तु चतुर्णामित्यादि। ततः प्राचामपीति द्वितीयपक्षे तु–गर्गाचतुर्थे यूनि मूलप्रकृत्यनन्तराभ्यामनिष्टोत्पत्तिः प्रसज्ते। पञ्चमे तु –मूलप्रकृत्यनन्तरयुवभ्यः। तथा नडाच्चतुर्थे यूनि पूर्ववद्द्वाभ्यामनिष्टोत्पत्तिः। पञ्चमे त्रिभ्यः। उपगोश्चतुर्थे वाच्ये तु मूलप्रकृतेरेकस्मादेवानिष्टोत्पत्तिः, न त्वनन्तरा पत्यवाचकात्। ततो जातेऽप्यत इञि रूपानिष्टाभावादित्यादि यथासंभवं तत्रोह्यमित्यर्थः।
