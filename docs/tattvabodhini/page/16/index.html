युटू। आदिर्ञिटुडवः इत्यत आदिः, षः प्रत्ययस्य इत्यतः प्रत्ययस्येति चानुवर्तते, तदाह-प्रत्ययाद्याविति। प्रत्ययाद्यविति किम् ?। वाचाटः। तेन वित्तश्चुञ्चुप्चणपौ इत्यत्र प्रत्ययादौ यकारो लुप्तनिर्दिष्टस्तेन चस्य नेत्संज्ञेति वक्ष्यति।
