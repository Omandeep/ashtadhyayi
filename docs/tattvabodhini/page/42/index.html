सुमद्रमिति। उत्तरपदार्थप्राधान्ये तु कुगतीति तत्पुरुषः—समृद्धा मद्राः सुमद्राः। दुर्यवनमिति। न चार्थाऽभावेनेह सिद्धिः, येन समस्यते तदीयार्थभावे तत्र समासस्वीकारात्। न चेह यवनानामभावो, येनार्थाऽभावे सूत्रे सर्वपश्चादिति भाष्यप्रयोगाच्चेत्यपि बोध्यम्। प्रत्यर्थमिति। वृत्तौ वीप्सान्तर्भावान्न द्विर्वचनम्। प्रतिशब्दस्येति। यत्त्वाहुः–ग्रामस्य वृक्षं वृक्षं प्रतीत्यत्र सापेक्षत्वेन समासाऽभावे
