दाम। बहुव्रीहिविशेषणत्वादेव सिद्धेऽन्तग्रहणं स्पष्टप्रतिपत्त्यर्थम्। सङ्ख्यादेरिति। समासनिर्देशेऽपि स्वरितत्वबलादेकदेश एवानुवर्तत इति भावः। अननुवृत्तेरिति। भाष्यकृताऽननुवर्ततत्वादिति भावः। एवं चोद्दामेत्यत्र दामहायनान्ताच्चेत्यस्याऽप्रवृत्त्या पूर्वोक्तेषु डाप्ङीब्नेषेधेषु प्राप्तेषु बहुराज्ञीवद्रूपत्रयं भवत्यतो व्याचष्टे—डान्बिषेधावपीति।
