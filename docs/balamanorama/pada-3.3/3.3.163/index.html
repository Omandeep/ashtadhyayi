BALAMANORAMAS

January 1, 0001

प्रैषातिसर्ग।लोट् चेति। पूर्वसूत्रोपात्तो लोट् चकारात्समुच्चीयते। कृत्यसंज्ञकाः प्रत्यया वक्ष्यमाणा, लोट् च प्रैषादिषु भवन्तीत्यर्थः। प्रैषे अतिसर्गे च कृत्यप्रत्ययमुदाहरति– भवता यष्टव्यमिति। भावे तव्यप्रत्ययः। लोटमुदाहरति– भवान्यजतामिति. ननु चकारेण लोटोऽनुकर्षणं व्य्रथं, प्रैषस्य विधिरूपतया, अतिसर्गस्य आमन्त्रणरूपतया च लोट् चेत्यनेनैव सिद्धेरित्यत आह– वकारेणेति। प्राप्तकाले यथा– गुरुणा भोक्तव्यम्। गुरुर्भुञ्जीत। भोजनं प्राप्तावसरमित्यर्थः। प्राप्तकाले च कृत्याश्चेत्युक्तौ तु निमन्त्रणादिष्वपि कृत्याः स्युः। अतः प्रैषादिग्रहणम्।
