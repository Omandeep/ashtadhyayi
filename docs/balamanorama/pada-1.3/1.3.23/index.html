BALAMANORAMAS

January 1, 0001

प्रकाशन। प्रकाशनं– स्वाभिप्रायाविष्करणम्। स्थेयः– विवादपदनिर्णेता। तिष्ठति = विश्राम्यति विवादपदनिर्णयोऽस्मिन्नित्यर्थे बाहुलके अधिकरणे अचो यत् इति यति [ईत्वे] स्थेयशब्दव्युत्पत्तेः। स्थेयो विवादस्थानस्य निर्मएतरि पुरोहिते इति मेदिनी। इह तु विवादपदनिर्णेतृत्वेनाध्यवसायो विवक्षितः। आख्या = अभिधानम्। प्रकाशनाख्यायां, स्थेयाख्यायां च वर्तमानात् स्थाधातोरात्मनेपदमित्यर्थः। प्रकाशने उदाहरति - गोपी कृष्णाय तिष्ठते इति। श्लाघह्नुङ्स्थाशपामिति संप्रदानत्वाच्चतुर्थी। स्थेये उदाहरति- - संशय्येति। कर्णादिष्विति विषयसप्तमी। फलितमाह– कर्णादीनिति।
