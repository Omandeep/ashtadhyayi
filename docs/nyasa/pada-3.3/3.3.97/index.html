NYASAS

January 1, 0001

स्वरार्थं वचनम् इति। न तु प्रत्ययार्थम्। स्त्रियां क्तिन् (3.3.94) इत्येव सिद्धत्वात्। नाप्यूडर्थम्; ज्वरत्वर (6.4.20) इत्यादिनैव सिद्धत्वात्। यौतेःइति। यु {मिश्रणेऽमिश्रणे च-धा।पा} मिश्रणे (धा।पा।१०३३) इत्यस्य।जवतेश्चइति।जुइत्यस्य सौत्रस्य (3.2.150) धातोः।दीर्घत्वञ्चइति। चकारादुदात्तत्वञ्च। क्तिन् सामान्यलक्षणेनैव सिद्धः।ल्यतेःइति।षोऽन्तकर्मणि(धा।पा।११४७) इत्यस्य।इत्त्वाभावोऽपि निपात्यतेइति।द्यतिस्यति(7.4.40) इत्यादिनेत्त्वं प्राप्नोति, ततस्तस्याभावो निपात्यते।सनोतेर्वाइति।षणु दाने(धा।पा।१४६४) इत्यस्य।जनसनखनाम्(6.4.42) इत्यात्त्वे कृते सातिरिति सिध्यतीति शेष-। यद्येवम्, किमर्थं निपातनमित्याह--स्वरार्()थमइत्यादि। असति हि निपातने सातिश्शब्द आद्युदात्तः स्यात्। अन्तोदात्तो यथा स्यादित्येवमर्थ निपातनम्।हन्तेर्हितनोतेर्वाइति। हेतिरिति यदा हन्तेः, तदा हकारस्यैत्त्वं निपात्यते, अनुनासिकलोपस्तुअनुदात्तोपदेश(6.4.37) इत्यादिनैव सिद्धः। यदा तुहि {गतौ बुद्धौ च-धा।पा} (धा।पा।१५७) इत्स्य, तदा गुणो निपात्यते। कीर्त्तिःइति। कृतं संशब्दने (धा।पा।१६५३) इत्यस्माच्चुरादिणिजन्तात् ण्यासश्रन्थो युच् (3.3.107) इति युचि प्राप्ते क्तिन्निपात्यते। उपधायाश्च (7.1.101) इतीत्त्वम् भवति, हलि च (8.2.77) इति दीर्घत्वम्॥
