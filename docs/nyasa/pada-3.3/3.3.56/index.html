NYASAS

January 1, 0001

चकारो विशेषणार्थः इति। विनापि तेन थाथादिसूत्रे (6.2.143) अप्रत्ययादित्यप्रत्ययस्यापि ग्रहणं स्यात्। अज्विधौ इत्यादि। उपसंख्यानशब्दस्य प्रतिपादनमर्थः। प्रतिपादनं तु कृत्यल्युटो बहुलम्(3.3.113) इत्येतदाश्रित्य कत्र्तव्यम्। अन्यस्तु पञ्चमी भयेन (2.1.36) वर्षप्रमाण ऊलोपश्चास्यान्तरस्याम् (3.4.32) इति निपातनाद्भयादीनां सिद्धिमिच्छति, न चासौ युक्ता; किमिदं भयवर्षशब्दयोर्नपुंसकलिङ्गयोरुच्चारणम्? इत् पुंल्लिङ्गयोः? इति सन्देहात्। वक्तयौइति। व्याख्येयावित्यर्थः। व्याख्यानं तु तदेव बहुलग्रहणमाश्रित्य कत्र्तव्यम्। जवः, सवः इति। जु (3.2.150) इत्येस्मात् सौत्राद्धातोः, षुञ् अभिषवे (धा।पा।१२४७) इत्येस्माच्चोत्तरसूत्रेणापि प्राप्तेऽज्भवति। स्वरे विशेषः– अपि हि स्तामाद्युदात्तं पदं स्यात्, अजन्ते सत्यन्तोदात्तत्वं भवति॥
