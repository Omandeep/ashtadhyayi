NYASAS

January 1, 0001

मर्यादावचने सति इति। वचनग्रहणमभिविधावपि यथा स्यात्। मर्यादाविशेष एवाभिविधिश्च। सैव मर्यादा यदा कार्येणाभिसम्बद्ध्यते तदाऽभिविधिरित्युच्यते, यदा तु न सम्बद्ध्यते तदा मर्यादेति। तत्रेह वचनग्रहणाद्विशेषो नाश्रीयते, तेन मर्यादाया उक्तिमात्रे भवति। विनापि वचनग्रहणेनाभिविधौ प्रतिषेधो न भवेत्; यस्माद्यत्राभिविधौ कार्यमिच्छति तत्राभिविधिग्रहणं वा करोति, यथा– आङ् मर्यादाभिविध्योः (2.1.12) इति#इ; वचनग्रहणं वा, यथा– आङ् मर्यादावचने (1.4.88) इति। तदभिविधावपि यथा स्यादित्येवमर्थं वचनग्रहणं क्रियते। अवरस्मिन् इति। अर्वागित्यर्थः। गन्तव्यःइति। भविष्यत्कातामध्वनो दर्शयति। तस्य यदवरम् इति। अनेनाध्वनो विभागं द्विः इति। एतेनापि क्रियाप्रतिबन्धाभावम्। भोक्ष्यमहे इति। भुजोऽनवने (1.3.66) इत्यात्मनेपदम्, चोः कुः (8.2.30) इति गकारः, खरि च (8.4.54) इति गकारस्य ककारः। अध्यैमहि इति। महिङ, आडजादीनाम् (6.4.72) इत्याट्, आटश्च (6.1.87) इति वृद्धिः, उपसर्गस्य (6.1.74) यणादेशः। अपिबाम इति। पाघ्रादिसूत्रेण (7.3.78) पिबादेशः, नित्यं ङितः (3.4.99) इति सकारलोपः। भोक्तास्महे इति। लुट्। तत्रच विशेषं वक्ष्यति इति। अनहोरात्राणामित्यनेन (3.3.137)॥
